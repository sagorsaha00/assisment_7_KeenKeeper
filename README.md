# KeenKeeper

KeenKeeper is a modern and responsive relationship tracking web application that helps users manage and monitor their personal connections. It allows tracking of interactions like calls, messages, and video chats through a clean timeline and visual analytics.

---

## 🚀 Features

- 🔄 **Global State Management (Context API)**  
  Manage and share timeline data across the entire app without prop drilling.

- 🔍 **Timeline Filtering System**  
  Filter interactions by type (Call, Text, Video) for better tracking and organization.

- 📊 **Responsive Pie Chart Analytics**  
  Visualize interaction data using a responsive pie chart for quick insights.

- 📱 **Fully Responsive Design**  
  Works smoothly on mobile, tablet, and desktop devices.

- 🔔 **Toast Notifications**  
  User-friendly alerts using React Toastify.

- ❌ **Custom 404 Page**  
  Handles unknown routes with a clean error page.
